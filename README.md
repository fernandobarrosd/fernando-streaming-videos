# Fernando Streaming Videos

## Sobre
Este projeto tem como intuito ser um local eu posso hospedar os meus vídeos,<br>
possibilitando as pessoas poderem assisti-los.<br>

- HTML5
- CSS3
- Typescript
- Vite
- Node

- Vercel
